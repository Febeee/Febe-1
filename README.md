# 人工智能项目四 - 作业仓库

## 项目概述
本仓库为《人工智能项目四》课程的作业存储库，用于记录和展示Jupyter Notebook基础实验代码，是课程作业的提交载体。

## 实验内容
1.  **环境搭建**：配置Python + Jupyter Notebook开发环境，完成Git与GitHub的基础配置
2.  **基础练习**：编写并运行 `jupyter_first_try.ipynb` 进行Python基础语法测试，完成代码的本地提交与远程仓库上传

## 文件结构
*   **`jupyter_first_try.ipynb`**: 主实验文件，包含完整代码、运行结果与实验记录
*   **`README.md`**: 项目说明文档（本文件），用于说明项目背景、运行环境与作者信息

## 运行环境
*   **操作系统**: Windows
*   **开发工具**: Jupyter Notebook / VS Code
*   **Python版本**: 3.x
*   **版本控制**: Git + GitHub
